The Language of
Neuron Architecture.

NeuroPy no es solo un wrapper. Es el lenguaje de interfaz diseñado por LogAI para controlar el núcleo Neuron, permitiendo el despliegue nativo de modelos como Cleo Nano.

Principios base del lenguaje. NeuroPy requiere la extensión .npy o .ny para ser interpretado correctamente por el core de Neuron.

import MyLLM.lib // Carga de librerías

Estructura de Creación (LLM)

create LLM {
  name model ("MyLLM")
  optimizer Adam(lr=0.001)
  save-model "path/to/model.ggfu"
}
start create LLM (
  save-data "log.json"
)

Lógica de Respuesta (ofm)

ofm MyLLM.lib( // Open For Model
  send(imf"MyLLM") // User Input
  response(@) // Model Response
end ofm
