Crea LLMs con la simplicidad de Python.

NeuroPy combina la arquitectura de alto rendimiento de Neuron con la flexibilidad de Python para democratizar el modelado de lenguaje masivo.

# Instalación rápida

pip install neuropy-llm


import neuropy

# Define un modelo en segundos

model = neuropy.LLM(layers="neuron-alpha")

model.train(dataset="custom_data")

Tecnología

Core de Neuron

Aprovecha la eficiencia de procesamiento de Neuron para manejar parámetros masivos sin degradar el rendimiento.

Sintaxis

Simplicidad Pythonic

Diseñado para que cualquier desarrollador de Python pueda construir un LLM sin ser un experto en álgebra lineal compleja.
